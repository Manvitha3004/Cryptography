1. INTRODUCTION
================================================================================

1.1 WHAT IS QUANTUM-SAFE DIGITAL TIME CAPSULE?
-----------------------------------------------

Quantum-Safe Digital Time Capsule is a secure messaging application that lets
you create encrypted messages that can only be opened after a specific date.
Think of it as a digital time capsule - you lock a message today, and it stays
locked until the unlock date you choose.

Key Features:
â€¢ Create encrypted time capsules with custom unlock dates
â€¢ Military-grade encryption protects your messages
â€¢ Digital signatures ensure message authenticity
â€¢ Multiple interfaces: web, command-line, and desktop
â€¢ Works offline - all data stored locally
â€¢ No cloud storage - complete privacy

1.2 WHO IS THIS FOR?
--------------------

Perfect for:
â€¢ Personal time capsules (letters to your future self)
â€¢ Business time-locks (scheduled information release)
â€¢ Legal documents with delayed disclosure
â€¢ Educational demonstrations of cryptography
â€¢ Anyone interested in secure messaging

1.3 HOW IT WORKS
----------------

Simple 4-step process:

Step 1: Generate Keys
       Create your unique cryptographic keys (one-time setup)

Step 2: Write Message
       Type the message you want to lock in the time capsule

Step 3: Set Unlock Date
       Choose when the message should become available

Step 4: Decrypt Later
       After the unlock date, decrypt and read your message

Behind the scenes:
- Your message is encrypted with advanced encryption
- The encryption key is locked with post-quantum cryptography
- A digital signature proves the message hasn't been tampered with
- Time-lock prevents decryption before the unlock date

8. TROUBLESHOOTING
================================================================================

8.1 COMMON PROBLEMS AND SOLUTIONS
----------------------------------

PROBLEM: "No keys found" error

Solution:
1. Generate keys first
2. Click "Generate Keys" button
3. Wait for confirmation
4. Try your operation again

---

PROBLEM: Cannot decrypt capsule - "Capsule locked"

This is NORMAL behavior, not an error!

Solution:
1. Check the unlock date
2. Wait until that date passes
3. For testing, create a new capsule with today's date

---

PROBLEM: Web interface won't open

Solutions:
1. Check if backend is running (python api.py)
2. Check if port 5000 is available
3. Check if frontend is running (npm start)
4. Try closing and restarting both servers
5. Clear browser cache

---

PROBLEM: "Signature verification failed"

This means the capsule was modified!

Solutions:
1. Don't manually edit capsule JSON files
2. Restore from backup if available
3. The capsule is likely corrupted - don't trust it

---

PROBLEM: GUI window doesn't open

Solutions:
1. Check if Python is installed: python --version
2. Check if tkinter is installed (should be included)
3. Try CLI instead: python capsule_main.py
4. Reinstall Python with GUI support

---

PROBLEM: Capsule list is empty

Solutions:
1. Create a capsule first (option 2)
2. Check if data/capsules/ folder exists
3. Check if you're in the right directory

---

PROBLEM: Application is slow

Solutions:
1. Check disk space (needs 100 MB minimum)
2. Close other applications
3. Reduce number of capsules (archive old ones)
4. Restart the application

9. FAQ (FREQUENTLY ASKED QUESTIONS)
================================================================================

Q1: Can I change the unlock date after creating a capsule?

A: No. The unlock date is cryptographically sealed in the capsule. This is
   by design - it's a time capsule! If you need a different date, create a
   new capsule.

---

Q2: What happens if I lose my keys?

A: You CANNOT decrypt your capsules without the keys. Always keep backups!
   The keys are in data/keys/user_keys.json.

---

Q3: Can someone else decrypt my capsules?

A: Not without your private keys. The encryption is military-grade. Keep
   your keys secret and secure.

---

Q4: Is my data sent to the cloud?

A: NO. Everything stays on your computer. This is 100% local, no internet
   needed after initial setup.

---

Q5: Can I edit a capsule after creation?

A: No. Capsules are immutable (can't be changed). This ensures integrity.
   Create a new capsule instead.

---

Q6: What's the maximum message length?

A: 10,000 characters (about 2 pages of text). This is a soft limit and can
   be increased if needed.

---

Q7: Can I create multiple capsules?

A: Yes! Create as many as you want. Each has its own unlock date and message.

---

Q8: What if my computer's clock is wrong?

A: The unlock check uses your system time. If your clock is wrong, the unlock
   times will be wrong too. Keep your system time accurate.

---

Q9: Can I use this for files and images?

A: Currently, only text messages are supported. For files, you'd need to
   encode them as text (Base64) or wait for future updates.

---

Q10: Is this really quantum-safe?

A: The current version uses classical cryptography (RSA/ECDSA) which is
   vulnerable to quantum computers. It's called "quantum-safe" because it's
   designed to be upgraded to true post-quantum algorithms (Kyber/Dilithium)
   in the future.

---

Q11: How much disk space do capsules use?

---

Q12: Can I share capsules with others?

A: Yes, you can share the encrypted capsule file. But they need YOUR keys
   to decrypt it. For secure sharing, exchange keys securely first.

---

Q13: What's the "QR capsule" feature?

A: Some versions generate QR codes of capsules for easy sharing. This is
   an advanced feature documented separately.

---

Q14: Can I run this on a server?

A: Yes, but you'll need to configure it for production use (add authentication,
   HTTPS, etc.). The current setup is for personal/local use.

---

Q15: Are there mobile apps?

A: Not currently. The web interface works on mobile browsers, but there's

10. GLOSSARY
================================================================================

AES (Advanced Encryption Standard)
   A military-grade encryption algorithm used to protect your messages.
   Very secure and fast.

Authentication
   Proving that something is genuine and hasn't been modified.

Base64
   A way to convert binary data to text so it can be stored in JSON files.

Capsule
   An encrypted message with a time-lock. Cannot be read until unlock date.

Ciphertext
   Your message after encryption. Looks like random gibberish.

Decryption
   Converting encrypted data (ciphertext) back to readable text (plaintext).

Digital Signature
   A cryptographic proof that a message is authentic and hasn't been tampered
   with. Like a wax seal on a letter.

ECDSA (Elliptic Curve Digital Signature Algorithm)
   The algorithm used to create digital signatures in this application.

Encryption
   Converting readable text (plaintext) to unreadable data (ciphertext) to
   protect it from unauthorized access.

JSON (JavaScript Object Notation)
   A text format for storing data. Used to save capsules and keys.

KEM (Key Encapsulation Mechanism)
   A method to securely share encryption keys. Used to protect the AES key.

Key (Cryptographic Key)
   A secret number used for encryption and decryption. Like a password, but
   much longer and more complex.

Metadata
   Information about the capsule (timestamp, unlock date) that's included
   in the signature.

PEM (Privacy Enhanced Mail)
   A format for storing cryptographic keys as text.

Plaintext
   Your message before encryption or after decryption. Readable text.

Post-Quantum Cryptography (PQC)
   Encryption methods designed to resist attacks from quantum computers.

Private Key
   The secret key that must be kept confidential. Used for decryption and
   signing.

Public Key
   The key that can be shared publicly. Used for encryption and verification.

RSA
   An encryption algorithm named after its inventors (Rivest-Shamir-Adleman).
   Used in this application for key encapsulation.

Signature Verification
   Checking if a digital signature is valid to ensure data authenticity.

Time-Lock
   A restriction that prevents decryption before a specified date.

Timestamp
   The exact date and time when a capsule was created.

Unlock Date
   The date when a time capsule becomes available for decryption.
